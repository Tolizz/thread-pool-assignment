# Thread Pool Implementation (Producer-Consumer)

Αυτό το αποθετήριο περιέχει την υλοποίηση ενός πλήρως λειτουργικού Thread Pool σε C, βασισμένου στο μοντέλο Παραγωγού-Καταναλωτή (Producer-Consumer) με χρήση POSIX Threads (Pthreads). Η ουρά εργασιών (FIFO) έχει
τροποποιηθεί ώστε να δέχεται δείκτες σε συναρτήσεις (πακέτα εργασίας), οι οποίες αναλαμβάνουν υπολογιστικό
φόρτο (υπολογισμός ημιτόνου γωνιών).

## Προαπαιτούμενα (Dependencies)
Για τη μεταγλώττιση και εκτέλεση απαιτείται ο `gcc` και το βασικό περιβάλλον ανάπτυξης του Linux (ή WSL).
Ο κώδικας κάνει χρήση της βιβλιοθήκης νημάτων (`pthread`) και της μαθηματικής βιβλιοθήκης (`m`).

## Αναπαραγωγή Πειραμάτων (Benchmarking)
Για τη μελέτη της απόδοσης και την εύρεση του βέλτιστου αριθμού Καταναλωτών (Consumers),
έχει αναπτυχθεί το αυτοματοποιημένο bash script `run_experiments.sh`. Το script μεταγλωττίζει τον κώδικα
αλλάζοντας δυναμικά τον αριθμό των Consumers (1, 2, 4, 8, 16), εκτελεί 20 επαναλήψεις για κάθε σενάριο,
και εξάγει τους χρόνους αναμονής των εργασιών στην ουρά.

### Βήματα Εκτέλεσης:

**1. Κλωνοποίηση του αποθετηρίου:**
```bash
git clone https://github.com/Tolizz/thread-pool-assignment.git
cd thread-pool-assignment

```


**2. Εκτέλεση του Script:**
Δώστε δικαιώματα εκτέλεσης στο αρχείο και τρέξτε το:

```bash
chmod +x run_experiments.sh
./run_experiments.sh

```

Μόλις ολοκληρωθούν τα πειράματα, τα δεδομένα (οι χρόνοι αναμονής σε μικροδευτερόλεπτα) θα εξαχθούν αυτόματα στο αρχείο `results.csv`, έτοιμα για εισαγωγή στο Excel και δημιουργία γραφημάτων.
